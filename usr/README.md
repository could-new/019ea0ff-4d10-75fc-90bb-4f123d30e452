# Annin

A beautiful, responsive Flutter application dedicated to Annin Tofu (Almond Jelly). It features an elegant UI to discover the history of the dessert and a simple, clean recipe guide.

## Features
- **Discover Screen**: A rich, beautifully formatted introduction to Annin Tofu.
- **Recipe Guide**: A step-by-step layout for preparing the perfect dessert at home.
- **Responsive Layout**: Designed to look stunning on mobile, tablet, and desktop viewports.

## Getting Started

1. Ensure you have the [Flutter SDK](https://flutter.dev/docs/get-started/install) installed.
2. Clone the repository and navigate to the project directory.
3. Run `flutter pub get` to install any dependencies.
4. Run `flutter run` to start the app on your preferred device or emulator.
